# Website for Bodhi Meditation Centre, Vistoria, Australia

The project uses [NextJS](https://nextjs.org/)

## Setup

The project was setup using the [NextJS Installation Guide](https://nextjs.org/docs/getting-started/installation).

## Developer

The following are the steps you need to setup your computer for development.

  * Install [NodeJS](https://nodejs.org/en)
  * Install [Yarn](https://classic.yarnpkg.com/en/docs/install)
  * Clone the repo `git clone git@github.com:sazep-bd/bodhimeditation.git`

The following commands are useful for development on your computer.
  * `yarn dev` - start the website on your computer
